# quant_project 视频录制指南

## 📋 录制前准备

### 1. 启动 Web 界面

```bash
cd /home/zjc/.openclaw/workspace/quant_project

# 方式一：使用启动脚本
./start_web.sh

# 方式二：直接运行
python3 -m streamlit run app.py

# 方式三（需要venv）
source venv/bin/activate
streamlit run app.py
```

启动后浏览器访问: **http://localhost:8501**

### 2. 准备录屏软件

#### 推荐工具

| 工具 | 平台 | 特点 |
|------|------|------|
| **OBS Studio** | Win/Mac/Linux | 免费、专业、功能强大 |
| **Camtasia** | Win/Mac | 商业软件、易上手 |
| **Screen Studio** | Mac | 简洁、效果专业 |
| **Win + G 录屏** | Windows 10/11 | 系统自带 |

#### OBS Studio 推荐设置

```
输出设置:
- 录像格式: MP4
- 编码器: x264
- 质量: High Quality
- 码率: 15-25 Mbps

视频设置:
- 分辨率: 1920x1080
- 帧率: 30 fps
```

## 🚀 快速录制命令

### OBS 录屏脚本 (Linux)

```bash
#!/bin/bash
# obs_recording.sh

# 启动Streamlit
cd /home/zjc/.openclaw/workspace/quant_project
streamlit run app.py &
SPID=$!

# 等待启动
sleep 5

# 提示用户
echo "Streamlit已启动: http://localhost:8501"
echo "请开始OBS录制..."
echo "按Ctrl+C停止Streamlit"

# 等待
wait $SPID
```

### 使用 FFmpeg 录制 (如果可用)

```bash
# 安装 FFmpeg
sudo apt install ffmpeg

# 录制屏幕 (X11)
ffmpeg -f x11grab -i :0.0 -t 00:00:30 -c:v libx264 -preset fast \
    video_demo_30s.mp4

# 录制特定窗口
ffmpeg -f x11grab -i :0.0+100,200 -t 00:00:30 \
    -c:v libx264 -preset fast video_demo.mp4
```

## 📞 常见问题

### Q: Streamlit 无法启动？
```bash
# 检查依赖
pip install streamlit pandas plotly scikit-learn

# 检查端口
lsof -i :8501

# 重新启动
pkill -f streamlit
streamlit run app.py
```

### Q: 录屏卡顿？
- 降低分辨率到 1280x720
- 降低帧率到 24fps
- 关闭不必要的后台程序
- 使用 SSD 存储录像

### Q: 旁白录制有回声？
- 使用指向性麦克风
- 关闭音响，使用耳机监听
- 在安静房间录制
