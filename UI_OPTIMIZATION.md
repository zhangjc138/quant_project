# quant_project 界面优化方案

## 用户使用流程分析

### 流程1: 找股票
1. 看热门板块/涨跌幅榜
2. 选行业/输入代码
3. 看评分和信号
4. 加入自选

### 流程2: 策略回测
1. 选中股票
2. 调整策略参数
3. 运行回测
4. 看结果/调参

### 流程3: 决策
1. 看综合评分
2. 看技术信号
3. 做决定

## 具体改进点

### 改进1: 仪表盘
- 今日热点板块
- 近期强势股
- 今日信号股
- 快速入口按钮

### 改进2: 选股页面
- 行业多选
- 信号筛选（买入/卖出/持有）
- 评分筛选
- 快速排序

### 改进3: 回测页面
- 策略参数可视化
- 实时显示收益
- 策略对比功能

### 改进4: 评分页面
- 一键跳转到评分
- 决策按钮（加入自选/回测/LSTM）

| 优先级 | 改进 | 说明 |
|--------|------|------|
| P0 | 仪表盘改造 | 添加热点股票 |
| P0 | 选股信号筛选 | 按信号过滤 |
