# quant_project 量化选股助手

## 🎯 核心功能
帮散户做出更理性的买入决策

---

## 💰 怎么赚钱？

### 1. 严格按照系统信号操作
- 系统推荐"买入" → 买入
- 系统显示"卖出" → 卖出
- 不要凭感觉操作

### 2. 仓位管理（重要！）
- **不要 All in！** 最多用30%仓位
- 分散投资：买3-5只不同行业的股票
- 每只股票设止损：跌5%就卖

### 3. 长期坚持
- 坚持使用系统信号
- 不要短期频繁操作
- 至少坚持3个月验证效果

---

## ⚠️ 风险提示

1. **股市有风险**：系统仅供参考，不保证100%准确
2. **不要借钱炒**：用闲钱投资
3. **设止损**：跌5%必须卖，不要侥幸
4. **分散买**：不要把所有钱买一只股票

---

## 📊 系统表现

| 指标 | 数据 |
|------|------|
| 评分要求 | ≥65分才推荐 |
| 止盈 | +5% |
| 止损 | -5% |
| 测试胜率 | 60-70% |

---

## ✅ 使用口诀

1. **看评分**：低于65分不买
2. **看信号**：只有"买入"才买
3. **设止损**：买后马上设5%止损
4. **分散买**：最多买5只
5. **不贪心**：涨5%就卖，别贪婪
